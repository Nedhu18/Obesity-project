# 🧠 Obesity Classification using Ensemble Learning

## 🚀 Overview
This project focuses on predicting obesity levels using machine learning techniques. It applies ensemble learning methods and feature selection to improve classification accuracy using real-world health data.

The goal is to build an interpretable and high-performing model that can support early health risk detection and preventive healthcare analysis.

---

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Machine Learning Algorithms
- Ensemble Methods (Random Forest, AdaBoost)
- Feature Selection (RFE)
- Matplotlib / Seaborn

---

## 📊 Problem Statement
Obesity is a major global health issue. Early classification of obesity levels can help in preventive healthcare. This project aims to classify obesity levels based on lifestyle and health-related attributes.

---

## ⚙️ Methodology

### 1. Data Preprocessing
- Handled missing values
- Encoded categorical variables
- Scaled numerical features

### 2. Feature Selection
- Recursive Feature Elimination (RFE) used to identify important features

### 3. Model Building
- Random Forest Classifier
- AdaBoost Classifier

### 4. Evaluation Metrics
- Accuracy
- Precision / Recall
- ROC-AUC Score

---

## 📈 Results
- Achieved **97% accuracy** using Random Forest
- ROC-AUC score of **0.99**
- RFE improved model efficiency and interpretability
